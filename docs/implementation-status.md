## 技术债务

### 高优先级

1. **全文搜索性能**
   - 位置：`NoteRepository.SearchAsync()`
   - 问题：使用 `LIKE '%keyword%'` 全表扫描
   - 修复：迁移到 SQLite FTS5 或 PostgreSQL tsvector

2. **向量化管道可靠性**
   - 位置：`NoteService.CreateAsync()` / `UpdateAsync()`
   - 问题：采用 `Task.Run` fire-and-forget 模式，高并发可能丢失任务
   - 修复：改用 `Channel<T>` + `BackgroundService` 队列式处理

### 中优先级

3. **Anthropic Embedding 占位实现**
   - 位置：`AnthropicEmbeddingProvider.cs`
   - 问题：调用时抛出 `NotSupportedException`
   - 修复：实现 Anthropic Embedding API 调用

4. **MCP SSE 类型未实现**
   - 位置：`McpService.cs`
   - 问题：仅支持配置，实际调用返回错误
   - 修复：实现 HTTP/SSE MCP 客户端

### 低优先级

5. **Skill Config 结构简单**
   - 位置：`Skill.cs`
   - 问题：仅 promptTemplate + systemPrompt，未使用 JSON Schema
   - 修复：扩展 Config 结构支持更复杂的参数定义

6. **MCP 进程无复用**
   - 位置：`StdioMcpClient.cs`
   - 问题：每次工具调用创建新进程
   - 修复：实现连接池或长连接复用
